# Screenshots

This folder contains screenshots of the Wine Quality Prediction & Advisor Dashboard, including:

* Dashboard Home Page
* Dataset Overview and Statistical Summary
* Wine Quality Distribution Analysis
* Correlation Heatmap Visualization
* Model Performance Comparison (Random Forest, SGD Classifier, and SVC)
* Accuracy Comparison Chart
* Confusion Matrix Analysis
* Feature Importance Visualization
* Wine Quality Prediction Interface
* Predicted Wine Quality Results
* Animated Wine Quality Scale (Low, Medium, High)
* Wine Advisor Mode Recommendations
* Downloadable Prediction Report


These screenshots demonstrate the functionality, machine learning workflow, data visualizations, prediction capabilities, and intelligent recommendation features of the application.

## Note:
All screenshots are captured from the running Streamlit application using the WineQT dataset.
