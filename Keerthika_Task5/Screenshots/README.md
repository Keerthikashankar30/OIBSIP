# Screenshots
The screenshots folder contains visual outputs of the Geo AI House Price Prediction System dashboard. These images help demonstrate the working of different modules in the application.

## Included Screenshots:

### Home Page
Displays the main dashboard with navigation options and project overview.

### Prediction Page
Shows the interactive input form where users can enter housing details and get predicted house prices in real-time.

### Analytics Page
Contains model evaluation results including R² score, MSE, Actual vs Predicted graph, and feature importance visualization.

### Geo Map Page
Displays the Folium-based geospatial heatmap showing house price distribution across different locations.

## Note:
All screenshots are taken from the running Streamlit application after uploading and processing the housing dataset.
