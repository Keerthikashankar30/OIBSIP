# Screenshots

This folder contains screenshots of the **NYC Airbnb Data Cleaning Dashboard** project.

## Included Visuals

The screenshots demonstrate different modules of the dashboard:

* Dataset Overview (Rows, Columns, Missing Values, Duplicates)
* Data Quality Report (Quality Score and Pie Chart)
* Missing Values Analysis (Bar Chart)
* Outlier Detection (Box Plot Visualization)
* Data Explorer (Search and Filter Feature)
* Cleaning Module (Cleaned Dataset Preview)
* Before vs After Cleaning Comparison

## Purpose

These screenshots help in visually understanding the functionality of the dashboard and showcase:

* Data cleaning process
* Interactive visualizations
* Dataset improvements after preprocessing
* Streamlit dashboard features

## Note

All screenshots are taken from the running Streamlit application after uploading the Airbnb dataset.
